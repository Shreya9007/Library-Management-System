# Library Management System using Java

## Project Overview
The Library Management System is a console-based Java application developed to manage basic library operations efficiently. The project demonstrates the implementation of Object-Oriented Programming concepts and CRUD operations using Java collections.

This application allows users to add, search, issue, return, view, and delete books through a menu-driven interface. A preloaded dataset of books is also included to simulate a real-world library environment.

---

## Features

- Add New Books
- View Available Books
- Search Books by ID
- Issue Books
- Return Books
- Delete Books
- Preloaded Book Dataset
- Book Categories
- Book Availability Status Tracking
